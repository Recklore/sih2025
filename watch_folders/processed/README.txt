================================
PROCESSED DOCUMENTS ARCHIVE
================================

This folder contains files that have been successfully processed and added to the database.

📁 Structure:
- Files are organized by date: YYYY-MM-DD/filename
- Original filename and timestamp are preserved

⚠️ Important:
- These are backup copies of processed files
- Deleting files from here does NOT remove them from the database
- To remove documents from the database, delete them from the watch folders BEFORE processing
- You can safely clean up old files here to save disk space

💡 Tip:
Check this folder to verify which documents have been processed successfully.
